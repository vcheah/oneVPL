Detailed Description

template<typename VideoParams, typename InitList, typename ResetList>
class oneapi::vpl::session< VideoParams, InitList, ResetList >

Base class to manage any session. User doesn't have ability to create instanse of this class. only specialized classes for decoder or encoder or VPP sessions must be used. General note: negative status is sent thought exceptions, warnings are delivered as a status codes.

Template Parameters
VideoParamsClass to manage Sesson paramers
InitListClass to manage extensio buffers for Init stage
ResetListClass to manage extensio buffers for Reset stage
